LibUTAU 216 For Qt
改写：Sinestr.
链接：GitHub

主要改写部分：
1. 数据类型全部用 Qt 的类实现，能用布尔型就不用整型；
2. 建立插件时可以选择是否加载音源库，对于无需读取oto的开发可以节省启动时间；
3. 输出文件时提供了删除原有音符属性的功能；
4. 完全与 Windows API 独立；

主要修正部分：
1. 原版注册原音设定时，优先级没有按照从上到下，会出现歌词对应采样错误的情况；
2. 原版的控制点没有考虑PBM；
3. 原版的先行声音、重叠校正，没有考虑前驱音符的曲速和当前音符的辅音速度；
4. 原版检测休止符时，没有考虑含空格的情形；
